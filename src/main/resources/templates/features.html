Vous ne pourrez plus vous passer de Docket


Commencez par sélectionner votre équipe dans le championnat. Vous ne pouvez sélectionner qu'une seule équipe à la fois, cependant rien ne vous empêche de générer plusieurs calendriers. Choisissez ensuite les types d'événements qui vous intéressent parmi les matchs de votre équipe, les tournois, les matchs fédéraux, les réunions de la fédération ou les autres événements.
Enfin cliquez sur générer le lien ou sur télécharger le calendrier.


Le fichier généré ainsi que le lien sont au format iCal. Ce format est un format standard d'événements pour les agendas. Il vous suffit de l'utiliser dans Google Agenda, iCal d'Apple, Outlook et bien d'autres outils de calendrier. Pour plus d'informations sur le format iCal, reportez vous à l'explication de Wikipedia.


Toutes les événements intégrés à votre agenda proviennent du site Volley 34. Les données sont mises à jour régulièrement. Reportez vous à l'aide de votre logiciel de calendrier pour intégrer le calendrier de ce dernier.


Une fois le calendrier généré vous pouvez envoyer le fichier ou le lien à d'autres personnes comme par exemple les membres de votre équipe. Cependant nous vous conseillons plutôt de créer un calendrier partagé dans votre logiciel préféré. En effet ce site étant entièrement gratuit il n'est pas prévu pour recevoir de nombreuses demandes. En partageant un calendrier (avec Google Agenda par exemple), vous faites travailler les serveurs de Google plutôt que ce site et vous pouvez rajouter des informations utiles à votre équipe.


Ce site est entièrement gratuit, 100% sans publicité garanti et aucune adresse email ne vous sera demandée !
